# 変わった計算ソフト

## 詳細
値を入力して足し算を行い、１０以上の値を入力すると掛け算を行う。  
  
## 必要なソフトウェア  
  * Python  
	   * テスト済み: 3.7〜3.10  
  * テスト環境  
	   * Ubuntu20.04
